About Network Protocols

File Transfer Protocol (FTP)

File Transfer Protocol (FTP) is a set of rules that allow files to be exchanged between computers on the Internet. Web developers commenly use FTP to transfer web page files from their computers to web servers. FTP is also used to download programs and files from other servers to indivisual computers.

More on File Transfer Protocol

Hypertext Transfer Protocol (HTTP)

HTTP is a set of rules for exchanging files such as text, graphic images, sound, video, and other multimedia on the Web. Web browsers send HTTP requestes for web pages and their associated files. Web servers send HTTP responses back to the web browsers.

More on Hypertext Transfer Protocol (HTTP)

Internet Protocol (IP)

IP is a set of rules that controls how data is sent between computers on the internet. IP routes a packet to the correct destination address. The packet gets successively forwarded to the next closest router (a hardware device designed to move netword traffic) until it reaches its destination.
